PlatyPS.Hosting

PlatyPS.Hosting is a PowerShell module that extends the Microsoft.PowerShell.PlatyPS v1 help authoring workflow. It provides commands to generate, render, and publish browser-friendly HTML documentation from PlatyPS-authored help.

Commands

Command Description
ConvertTo-Aliases Renders the aliases for a command as an HTML section.
ConvertTo-CodeBlock Converts markdown fenced code blocks to HTML <pre><code> elements.
ConvertTo-Description Renders the description for a command as an HTML section.
ConvertTo-Examples Renders the examples for a command as an HTML section.
ConvertTo-HtmlEncoded HTML-encodes a string by replacing special characters with HTML entities.
ConvertTo-HugoFormat Converts PlatyPS-generated Markdown files to Hugo-compatible format.
ConvertTo-InputsOutputs Renders an INPUTS or OUTPUTS section as HTML.
ConvertTo-Notes Renders the NOTES section of a command as an HTML section.
ConvertTo-Parameters Renders the parameters for a command as a detailed HTML section.
ConvertTo-RelatedLinks Renders the related links for a command as an HTML section.
ConvertTo-Synopsis Renders the SYNOPSIS section of a command as HTML.
ConvertTo-Syntax Renders the SYNTAX section of a command as HTML.
Export-HtmlCommandHelp Custom HTML renderer for PlatyPS CommandHelp objects.
New-HelpPage Builds the complete HTML page for a single command.
New-ModuleHelp Generates all help artifacts for a loaded module in one step.
New-ModuleIndexPage Builds the HTML module index page for a set of commands.
Publish-ModuleHelp Publishes PowerShell module HTML help files to an IIS web server.
Resolve-HtmlTheme Loads a theme psd1 and merges it over the built-in defaults.
Write-Section Wraps HTML content in a styled <section> element.
