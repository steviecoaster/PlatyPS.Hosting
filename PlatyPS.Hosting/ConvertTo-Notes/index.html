ConvertTo-Notes

SYNOPSIS

Renders the NOTES section of a command as an HTML section.

SYNTAX

__AllParameterSets

ConvertTo-Notes [[-Help] <CommandHelp>]

ALIASES

None

DESCRIPTION

Accepts a PlatyPS CommandHelp model object and returns an HTML <section> block with each non-blank line of the notes text wrapped in a <p> tag. Returns an empty string if the command has no notes.

EXAMPLES

EXAMPLE 1

$importParams = @{
    Path = '.\docs\MyModule\Get-Widget.md'
}

$help = Import-MarkdownCommandHelp @importParams

ConvertTo-Notes -Help $help

Returns an HTML notes section for the Get-Widget command.

PARAMETERS

-Help

The PlatyPS CommandHelp model object to extract notes from.
