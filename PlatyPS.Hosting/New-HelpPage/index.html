New-HelpPage

SYNOPSIS

Builds the complete HTML page for a single command.

SYNTAX

__AllParameterSets

New-HelpPage [[-Help] <CommandHelp>] [[-AllCommands] <string[]>] [[-Theme] <hashtable>]

ALIASES

None

DESCRIPTION

Orchestrates all ConvertTo-* renderers to produce a full self-contained HTML page for one command, including the themed CSS variables, a page header, a sidebar with navigation links to other commands in the module, and all content sections such as synopsis, syntax, aliases, description, examples, parameters, inputs, outputs, notes, and related links.

EXAMPLES

EXAMPLE 1

$importParams = @{
    Path = '.\docs\MyModule\Get-Widget.md'
}

$help = Import-MarkdownCommandHelp @importParams

$pageParams = @{
    Help        = $help
    AllCommands = @('Get-Widget', 'Set-Widget')
    Theme       = Resolve-HtmlTheme
}

New-HelpPage @pageParams

Returns the complete HTML string for the Get-Widget command page with sidebar navigation and the default theme applied.
